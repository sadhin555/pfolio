### Run Locally

To run **Pfolio** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/sadhin555/pfolio.git
```

Windows:

```bash
git clone https://github.com/sadhin555/pfolio.git
```
